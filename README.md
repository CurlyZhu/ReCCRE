# Understanding user-LLM conversations with the ReCCRE (Request Content, Context, Roles, and Expressions) dataset
Code and data of EACL 2026 paper "Show or Tell? Modeling the evolution of request-making in Human-LLM conversations".

*We are not able to fully publicize the data/code for now due to a follow-up work in submission. We will hopefully be able to publish them by May 2026. Meanwhile, please feel free to email the 1st author directly to discuss and/or request these resources!*

- Annotated dataset
  - WildChat
  - Stanford Politeness
- Prompts
- Data analysis script
